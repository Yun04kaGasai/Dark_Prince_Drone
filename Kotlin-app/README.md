# Drone BLE Control (Kotlin)

Android-приложение для управления дроном по BLE.

## Интерфейс
- Центральная карточка с минималистичным неоморфным чёрно-белым дизайном.
- Боковое меню: `Аппараты`, `Управление`, переключение темы.
- Поддержка тем:
- системная (по устройству)
- светлая (вручную)
- тёмная (вручную)

## Параметры BLE
- Имя устройства: `DarkPrinceDrone`
- UUID сервиса: `6e400001-b5a3-f393-e0a9-e50e24dcca9e`
- UUID характеристики команд: `6e400002-b5a3-f393-e0a9-e50e24dcca9e`

## Пины моторов
Приложение отправляет команды скорости на пины:
- `4`
- `5`
- `2`
- `3`

## Управление тягой
- Кнопка `Вверх` увеличивает общую тягу на шаг.
- Кнопка `Вниз` уменьшает общую тягу на шаг.
- Текущее значение отображается как `Тяга: N`.
- Для всех 4 моторов отправляется одинаковая скорость.

## Протокол команд
Каждая команда отправляется как UTF-8 текст, заканчивающийся `\n`:
- `ARM`
- `DISARM`
- `STOP`
- `M,<pin>,<speed>`, где скорость в диапазоне `0..255`

Пример:
`M,4,120`

## Использование
1. Прошейте `Drone.ino` (или `Assets/Drone.ino`) в ESP32.
2. Откройте `Kotlin-app` в Android Studio.
3. Соберите и запустите приложение на Android-смартфоне с BLE.
4. Выдайте приложению Bluetooth-разрешения.
5. В разделе `Аппараты` найдите `DarkPrinceDrone` и подключитесь.
6. Перейдите в раздел `Управление`, нажмите `ARM`, затем управляйте кнопками `Вверх`/`Вниз`.
7. Для остановки используйте `STOP`, затем `DISARM`.

## Безопасность
- Всегда тестируйте сначала со снятыми пропеллерами.
- Перед отключением питания используйте `STOP` и `DISARM`.
