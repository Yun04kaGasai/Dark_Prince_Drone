# Dark_Prince_Drone
  Доступная, модульная и полностью настраиваемая платформа на базе массового микроконтроллера ESP32 от мира дронов.
# КОМПОНЕНТЫ
- ESP32
- Драйвер двигателя TB6612FNG
- Двигатели 614 (обычно сразу идут с лопастями)
- Модуль GY-87
- Плата зарядки (Можно взять любую, но см. ниже)
- - Рекомендую взять TP4056
- Аккумулятор 602040 на 420-450 мАч (3.7 В)
